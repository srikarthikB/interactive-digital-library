About the Project

The Interactive Digital Library is a prototype designed to enhance digital reading experiences by integrating AI-generated summaries, visual representations, and optional audio narration.

⚠ Copyright & Fair Use Notice

This project does not host or distribute copyrighted books. All full texts are accessed through external public platforms such as Archive.org. The AI-generated summaries, visuals, and audio content provided here are intended solely as educational pre-reading aids under fair use principles.

Instead of hosting copyrighted books directly, the platform ethically redirects users to publicly accessible sources such as Archive.org, while offering AI-powered pre-reading aids to improve comprehension and engagement.

This project was developed as part of an academic Design Thinking & Innovation assignment, focusing on accessibility, ethical content use, and user-centered design.

Key Features

  • AI-generated text summaries for books
  • Optional audio narration using text-to-speech
  • AI-generated visual scene summaries
  • Searchable and responsive book library
  • Ethical redirection to external reading sources
